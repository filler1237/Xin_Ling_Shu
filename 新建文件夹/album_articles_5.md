一

证件类

以下列出的证件资料

和相关资料都非常重要！

忘带的话可能会比较麻烦哦

为了保证人在证在

建议大家把所有资料

统一放在一个文件袋

//

新生都要带上

1、《录取通知书》原件

2、《中学学籍档案》

注：可以在入校后自行送达或者毕业中学统一邮寄。如果选择自行送达学校，请在“新生档案标签”上注明姓名、学号及学院等信息，并张贴于档案袋封面左上角，开学后上交

3、身份证原件：出门必备！最好能随身带着

4、身份证（正反面）A4复印件1份：需要签上自己的姓名

5、本人近期正面免冠一寸彩色证件照片10张：用于办理各类入学手续和新生体检

6、省外党员带上《中国共产党党员组织关系介绍信》原件，省内党员新生由原党组织直接在网上操作转接

7、团员带上团籍档案（含入团志愿书、团员证）。未在广东智慧团建系统报到的团员，还需上交《中国共产主义青年团团员组织关系介绍信》原件

8、符合条件的广东籍新生带《广东省公民兵役证》原件及复印件

打算迁户口的同学

办理户口迁移的同学，请携带以下材料。报到后一个月内，将相关资料递交所在学院。

1、广东省外新生带《户口迁移证》原件及复印件1份；广东省内深圳市外新生带本人户口簿原件及本人页和扉页复印件一份

2、《录取通知书》复印件1份

3、身份证（正反面）A4复印件1份

4、深圳身份证数码照相回执1张、一寸白底彩照3张

申请绿色通道/助学贷款的同学

因家庭经济突发经济困难需缓交学费或需通过生源地信用助学贷款进行学费缴纳的学生，记得准备好：

1、《深圳技术大学2024-2025学年家庭经济困难新生入学“绿色通道”（缓交学费）申请表》（报到时于“绿色通道”处领取）

2、《深圳技术大学家庭经济困难学生认定申请表》

二

床上用品类

深技大宿舍为上床下桌

两连体中梯的床

床板尺寸为

W1920*D840*H10（mm）

学校的床只有一块原始的木床板

想要睡软和舒适的被窝

就要靠自己动手准备好

各种床上用品啦

//

1、床垫：又厚又软的床垫是保证有好睡眠的基础哦，建议购买1.9m*0.85-0.9m的

2、床单、枕头、被子：基础三件套，被子建议可以根据需求自备空调被、大棉被应对不同的季节

3、凉席：夏天省电必备神器

4、床帘：拥有床帘就拥有了自己的私密空间。不同宿舍区的尺寸不一样，建议等开学之后住宿舍再买。

5、蚊帐：有了床帘其实也可以考虑买蚊帐，毕竟床帘一直挂着会闷，蚊帐可以透透气也可以挡蚊子

6、眼罩、耳塞：这些根据自己的实际情况准备，如果自己的睡眠质量不太好的话建议多备几套

三

电器类

要想宿舍有家一样的感觉

除了床上用品

不要忘了准备一些

实用的小功率电器

//

1、手机配件：充电头+耳机+数据线+移动电源要带齐

2、充电台灯：以备不时之需

3、电脑：开学会进行网上选课和抢课，有自己的电脑会方便很多，当然没带的话可以到学校图书馆借用

4、排插：宿舍插座一般是在每个床位的桌子下方，一个够长的排插能解决所有烦恼

5、吹风机：宿舍内没有配备吹风机，准备一个小功率吹风机是比较有必要的。（PS：宿舍用电功率超过1200W容易断电）

6、U盘：方便报告展示和拷贝上课PPT

7、可充电小风扇：停电救星，宿舍有时候会停电

8、网线：非必带，在校内或附近都可以购买

四

生活类

衣裤鞋袜、洗漱用品等

常见生活必需品一定要带齐

但不用一次性带很多

以免造成不必要的负担

还有一些日常用品是必不可少的

考试君整理出了以下

使用频率较高的物品

//

1、洗漱用品：牙膏、牙刷、毛巾、洗发液、沐浴露、毛巾

2、洗衣用品：水桶、脸盆、衣架、晾衣杆、洗衣液等

3、垃圾桶、垃圾袋

4、拖鞋、洞洞鞋：雨天出行必备

5、小锁：贵重物品上锁，有备无患

6、杯子：建议宿舍里放一个，方便冲麦片等食物

7、粘钩：挂书包、毛巾等都需要用到，最好多准备几个

8、收纳盒、收纳筐：可以放置各种小物件，防止要用找不到

9、晴雨伞：深圳多雨天，晴天时太阳毒辣，一把晴雨伞既可以遮阳，又可以挡雨

10、刀类：水果刀、指甲刀、剪刀、剃须刀

11、清洁工具：洗洁精、拖把、扫把、抹布、垃圾铲等

11、其他：抽纸、梳子、镜子、小夹子、收纳盒等

五

学习类

学习是大学生活中很重要的一部分

要准备好这些学习用品

来迎接和高中完全不同的大学课堂

//

1、背包：带一个好看又实用的，建议背包选既能装书又能放水杯的

2、笔记本、草稿本：大小厚薄不同的笔记本都多备几个，或者买活页本

3、书立、笔筒：保持书桌整齐干净才有学习的动力

4、文件夹、订书机、回形针：整理归类各种资料都需要用到

5、签字笔、彩色笔、2B铅笔、橡皮

6、双面胶、透明胶、胶水等

六

医药类

虽然说深技大有校医院

附近也有药店

但常备这些药品

关键时候会派很大用场

//

1、创可贴：小磕小碰最少不了它了

2、外伤用药：医用酒精、医用棉签、碘酒、红药水、纱布等受伤能帮你及时处理伤口

3、驱蚊用品：花露水、清凉油、风油精等驱蚊止痒

4、止痛药、防中暑药、防腹泻药：这些都是按自己的实际情况准备，有需要的买上备着预防意外情况

5、体温计
